# Random Shapes CLI

CLI tool for drawing randomly orientated and colored shapes with turtle graphics

## Usage

`python main.py <number of shapes> <shape>`

### Help

`python main.py -h`  
`python main.py --help`

### Squares Example

`python main.py -s 32`  
`python main.py --squares 999`
